### 📧 Contact Form Setup (Optional)

#### To Enable Email Notifications:
1. **Get Gmail App Password:**
   - Go to: https://myaccount.google.com/security
   - Enable 2-Step Verification
   - Go to "App passwords"
   - Generate password

2. **Create .env file in C:\WORK\RCA:**
   ```
   EMAIL_USER=your-email@gmail.com
   EMAIL_PASSWORD=your-16-char-app-password
   ```

3. **Restart server** (Ctrl+C, then `npm start`)

#### Test Contact Form:
1. Go to: http://localhost:3000/contact
2. Fill out form and submit
3. Check email for confirmation

### ⚡ Common Issues & Solutions

**"Port 3000 already in use"**
- Stop other Node.js processes: `taskkill /F /IM node.exe`
- Or use different port in server.js

**"Contact form not sending"**
- Ensure .env file is created
- Check Gmail App Password is correct
- See EMAIL_CONFIG.md for detailed help

**"Styles not loading"**
- Clear browser cache: Ctrl+Shift+Delete
- Restart server: Ctrl+C, then `npm start`

**"Page not found"**
- Check URL spelling
- Ensure server is running
- Check console for errors
